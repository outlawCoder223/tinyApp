# TinyApp Project

!["screenshot description"](https://github.com/rancewcampbell/tinyApp/blob/master/docs/home_page.png?raw=true)

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly). TinyApp keeps tabs on each URL a user creates by tracking how many visits their short URL has been visited and logs the details of each visit.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
